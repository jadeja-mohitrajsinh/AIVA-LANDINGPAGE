Introduction

AIVA (Advanced Intelligent Virtual Assistant) is a powerful AI-powered desktop application that combines natural language processing, voice interaction, and system control capabilities. Built with Electron.js for the frontend and Python FastAPI for the backend, AIVA provides a seamless experience for users to interact with their computer through natural language commands.

AI-Powered

Advanced natural language processing using Groq API for intelligent responses and actions

Voice Control

Hands-free operation with voice input and text-to-speech capabilities

System Control

Control system settings, launch applications, and manage files through voice commands

Smart Assistant

Context-aware responses and intelligent task automation

Key Features:
  • Natural language processing with Groq API integration
  • Voice input and text-to-speech capabilities
  • System control functions (volume, brightness, app launching)
  • File management and media control
  • Weather and currency conversion utilities
  • Note-taking and task management
  • Real-time chat with persistent history
  • Dark/light mode support

API Reference

Note:

AIVA uses our company's enterprise API service, so you don't need to configure any API keys. The service is automatically managed and optimized for your needs.

POST /api/chat

Process chat messages and generate AI responses

Request Body

Parameter Type Description
message* string The user's message
toolsoptional object Additional tools and options
Example Request
curl -X POST http://localhost:8000/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "message": "What's the weather like?",
    "tools": {
      "isVoiceInput": false
    }
  }'

Response

{
  "success": true,
  "data": {
    "messages": {
      "user": {
        "type": "user",
        "message": "What's the weather like?"
      },
      "assistant": {
        "type": "ai",
        "message": "Let me check the weather for you..."
      }
    },
    "timing": {
      "total": 1.23,
      "client_init": 0.1,
      "analysis": 0.3,
      "function_calls": [...],
      "final_response": 0.5
    }
  }
}
POST /api/speech

Process voice input and convert to text

Request Body

Parameter Type Description
audio_data* string Base64 encoded audio data
Example Request
curl -X POST http://localhost:8000/api/speech \
  -H "Content-Type: application/json" \
  -d '{
    "audio_data": "base64_encoded_audio_data"
  }'

Response

{
  "success": true,
  "data": {
    "text": "Transcribed text from audio"
  }
}

Troubleshooting

Voice Recognition Issues

  • Check microphone permissions
  • Ensure proper audio input device is selected
  • Check internet connection for Google Speech API

AI Response Issues

  • Verify Groq API key is valid
  • Check internet connection
  • Review server logs for errors

System Control Issues

  • Check application permissions
  • Verify system settings access
  • Review function execution logs

Getting Help

Log Locations

  • Frontend logs: Check browser console
  • Backend logs: Check server.log and api_handler.log
  • System logs: Check application logs in your OS
